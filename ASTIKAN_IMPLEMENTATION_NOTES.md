# Astikan Health Next.js Website Update

This build expands the uploaded Astikan healthcare content into a modern, responsive Next.js website.

## What changed

- Added a full `data/pdf-content.ts` content layer generated from the uploaded 160-page Astikan PDF.
- Every sitemap page now has a detailed animated long-form content system.
- Added page-map navigation, animated checklist widgets, narrative cards, gradient statement panels, CTA strips, visual healthcare illustrations, and sticky desktop content navigation.
- Preserved the Astikan gradient visual direction: red, violet, dark blue.
- Used the uploaded Astikan logo in header and footer.
- Added AOS and GSAP-driven reveal animations.
- Kept all pages responsive for desktop, laptop, tablet, and mobile.
- Added a desktop mega menu for the complete sitemap.

## Run locally

```bash
npm install
npm run dev
npm run build
```
